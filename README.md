# DigimonRPG Discord Update Bot

디지몬RPG 업데이트 공지에서 **월드 버프 변경** 정보를 자동으로 Discord 채널에 전송하는 봇입니다.

## 기술 스택

| 항목 | 내용 |
|------|------|
| Language | Python 3 |
| Discord 라이브러리 | discord.py 2.4.0 |
| 크롤링 | requests, BeautifulSoup4, lxml |
| 호스팅 | Render (Web Service) |

## 기능

- 매주 수요일 00:00 KST 월드 버프 정보 자동 전송
- 이전과 내용 동일 시 자동 전송 스킵 (MD5 해시 비교)
- 30분마다 새 업데이트 공지 감지 및 자동 전송
- `!월드버프` 명령어로 즉시 조회

## 명령어

| 명령어 | 설명 |
|--------|------|
| `!월드버프` | 최신 업데이트의 월드 버프 변경 정보 즉시 출력 |

## 크롤링 대상

- 대상 사이트: `https://www.digimonrpg.com/Pages/Notice/Update`
- 추출 항목: 최신 게시글의 **게임 내 변경사항 > 월드 버프 변경** 섹션

## 자동 전송 주기

| 트리거 | 조건 |
|--------|------|
| 주간 자동 전송 | 매주 수요일 00:00 KST |
| 신규 공지 감지 | 30분 주기 폴링, 새 글 등록 시 즉시 전송 |
